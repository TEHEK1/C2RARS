Примеры использования C2RARS для преобразования C кода в RARS-совместимый RISC-V assembly.


## Как использовать

### Шаг 1: Скомпилировать пример

```bash
cd /Users/kamir22/Kurs/build
./src/cli/c2rars -i ../examples/02_arithmetic.c -o 02_arithmetic.asm
```

### Шаг 2: Открыть в RARS

1. Запустить RARS
2. File → Open → выбрать `arithmetic.asm`
3. Нажать Run → Assemble
4. Нажать Run → Go

### Шаг 3: Увидеть результат

Вывод появится в консоли RARS.

---

## Список примеров

### 01_hello.c - Hello World
Простейший пример вывода строки.

**Демонстрирует:**
- Базовый вывод текста
- Использование RARS syscall `print_string`

**Ожидаемый вывод:**
```
Hello, RARS!
```

---

### 02_arithmetic.c - Арифметические операции
Демонстрация базовых математических операций.

**Демонстрирует:**
- Сложение, вычитание, умножение
- Вывод чисел
- Форматированный вывод

**Ожидаемый вывод:**
```
Arithmetic Operations Demo
==========================
a = 15, b = 27
a + b = 42
a - b = -12
a * b = 405
```
